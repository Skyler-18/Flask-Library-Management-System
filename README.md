# LIBRARY MANAGEMENT SYSTEM

## Try it out yourself
[Check out the live app here](https://flask-library-management-system.onrender.com)

## Steps to run locally
1. ```bash
   python -m venv venv
   ```
2. ```bash
   pip install -r requirements.txt
   ```
3. ```bash
   venv\Scripts\activate
   ``` 
4. ```bash
   flask run
   ```
